# SkillForge - Online Course Platform

## What you received
The online site is a demo running with local data. This code is the complete, production-ready application. To make it real, you need to connect your own services by setting the environment variables.

## Requirements
- Node.js 18+
- Accounts for: Supabase, Stripe, SendGrid, Anthropic, Google Cloud (Gemini, OAuth), Apple Developer.
